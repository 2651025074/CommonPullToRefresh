# CommonPullToRefresh
Widget with pull to refresh for Android,support loadMore for ListView and RecyclerView.

* 下拉刷新支持大部分`view`：`ListView`、`ScrollView`、`WebView`等
* 加载更多目前只支持`ListView`和`ScrollView`
